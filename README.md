# 产品中心图片更新说明

当您替换 `fenlei-chanp-jpg` 文件夹后，需要更新产品图片映射，以确保产品中心页面能够正确显示新的图片。

## 更新步骤

1. **打开终端**
   - 在Mac上，您可以使用Spotlight搜索"终端"并打开它。
   - 在Windows上，您可以使用搜索功能找到"命令提示符"或"PowerShell"。

2. **导航到网站目录**
   - 在终端中输入以下命令，然后按Enter键：
     ```
     cd /Users/liuhongwei/Desktop/网站设计/网页
     ```

3. **运行更新脚本**
   - 在终端中输入以下命令，然后按Enter键：
     ```
     python3 update_images_map.py
     ```

4. **等待脚本执行完成**
   - 脚本会扫描 `fenlei-chanp-jpg` 目录，生成新的产品图片映射。
   - 执行完成后，脚本会显示找到的产品数量。

5. **刷新产品中心页面**
   - 打开浏览器，访问产品中心页面：http://localhost:8007/product.html
   - 刷新页面，即可看到更新后的图片。

## 脚本功能说明

`update_images_map.py` 脚本会：
1. 递归扫描 `fenlei-chanp-jpg` 目录中的所有子目录
2. 找到包含图片的目录，并提取其中的图片路径
3. 生成新的 `product-images.json` 文件，包含所有产品的图片映射
4. 保存映射文件到网站目录

## 注意事项

- 确保您的系统已安装Python 3
- 确保 `fenlei-chanp-jpg` 文件夹的结构正确，包含产品图片
- 执行脚本时，确保终端的当前目录是网站目录
- 执行完成后，记得刷新产品中心页面以查看更新后的图片
